ERP Finance Dashboard

Overview

This interactive Power BI dashboard visualizes key financial KPIs from SAP S/4HANA data — including Profit & Loss, cost centers, and revenue trends. It enables real-time financial monitoring and supports business decision-making through drill-down analysis.

Tools & Technologies

  • Power BI (Desktop + Service)
  • Excel / CSV data exports from SAP FICO
  • DAX, Power Query, and Data Modeling
  • GitHub Pages for deployment
